# Tradeco

## Link del proyecto publicado

https://tradeco506.com/

## Descripcion

Sitio web para Tradeco Costa Rica. Presenta catalogo de pisos SPC y LVT, marcas, informacion de contacto, formularios y canales directos de comunicacion.

## Tecnologias usadas

- Astro
- React
- Tailwind CSS
- Sass
- Swiper
- Formspree
- Iconoir
- Astro Sitemap

## Requisitos

- Node.js 18+
- npm
- Endpoint activo de Formspree para el formulario de contacto

## Retos tecnicos

- Construir catalogos dinamicos para pisos SPC y LVT con rutas individuales.
- Integrar carruseles y modales para mostrar productos de forma visual.
- Configurar sitemap y metadata para un sitio comercial publico.
- Conectar formulario, WhatsApp, email y mapa de Google en una experiencia de contacto completa.
